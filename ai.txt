# Dev Workspace

Unified development workspace for Mark Constable's projects, organized as a Git submodule parent repository.

## Purpose

Single-command workspace setup (`git clone --recurse-submodules`) while keeping all projects independently clonable for others.

## Projects

- [aitxt](https://raw.githubusercontent.com/markc/aitxt/main/ai.txt) - ai.txt specification standard (aitxt.ing)
- [appmesh](https://raw.githubusercontent.com/markc/appmesh/main/ai.txt) - Desktop automation via D-Bus/OSC
- [netserva](https://raw.githubusercontent.com/netserva/monorepo/main/ai.txt) - Infrastructure platform (Laravel + Filament)
- [rentanet](https://raw.githubusercontent.com/markc/rentanet/main/ai.txt) - Static business website (renta.net)
- [spe](https://raw.githubusercontent.com/markc/spe/main/ai.txt) - PHP 8.5 micro-framework tutorial

## Architecture

- **Submodules**: Each project is an independent Git repo
- **Unified server**: `php -S localhost:8000 index.php` serves all projects
- **Cross-pollination**: Shared CSS framework (base.css/site.css) across projects

## Related

- `~/.ns/` - Private NetServa operations workspace
- `~/.rc/` - Public dotfiles repository
